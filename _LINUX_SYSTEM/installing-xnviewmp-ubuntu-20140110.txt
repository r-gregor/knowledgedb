How To Install XnViewMP In Ubuntu

   Hi guys,

   In this tutorial I will show you how to install  a cross-platform media browser, viewer and
   converter in your Ubuntu machine. It is called XnViewMP and is compatible with more than
   500 formats.

   1. Open a new terminal (CTRL+ALT+T)

   2.  Add the ppa to your software sources with the following command.
sudo add-apt-repository ppa:dhor/myway

   3.  Update the sources list with the following command.
sudo apt-get update

   4.  Run the following command to install XnViewMP.
sudo apt-get install xnview
