   If you’re managing Debian or Ubuntu servers, probably, you may use dpkg or apt-get commands often.
   These two commands are used to install, remove, update packages.

   In this brief tutorial, let us see how to check if a package is installed or not in DEB based
   systems.

   To check whether a particular package for example firefox, is installed or not using command:
dpkg -s firefox

   Also, you can do the same using dpkg-query command. This command displays the decent output, and
   ofcourse, you can wild cards too.
dpkg-query -l firefox

   You can also filter through grep to get results for the exact package you need. For example, I want
   to see which gcc packages are already installed on my system using dpkg command:
dpkg --get-selections | grep gcc
