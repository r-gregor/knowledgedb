   How can I learn data structures effectively?

   Originally Answered: What are best ways for learning data structures?

   The best way to lean data structures is to solve as many problems on each data structure. Start with
   simpler data structures like arrays, linked lists, stacks, etc. and then move to more complex ones
   like trees, graphs, tries.

   Set goals for yourself, lets say, to solve as many questions on arrays first week. When you read the
   question, think of ways you would use to solve the problem and how you will code the solution. If you
   are unable to figure out the solution, then look at the algorithm description to get a basic logic of
   the solution and then try coding yourself. In case, you are unable to solve the problem, then look at
   the solution and make sure you understand the solution completely, for all possible test cases. You
   should understand how the data structure should behave given any input, be it an edge case or a
   regular input.
