Instead of creating a buffer of size 256, or 1024 bytes and saying "It outh to be enough (I hope!)", we can use
snprintf() function to get the number of characters that would have been written including the '\0' terminator,
and than malloc that same amount of memory for formated string:

int needed = snprintf(NULL, 0, "user=%s id=%d", name, id);
char *buffer=malloc((size_t)needed + 1);
snprintf(buffer, (size_t)needed + 1, "user=%s id=%d", name id);
