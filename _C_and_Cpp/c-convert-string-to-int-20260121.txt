Convert String to int in C
23 Jul, 2025

   In C, we cannot directly perform numeric operations on a string representing a numeric value. We
   first need to convert the string to the integer type. In this article, we will discuss different ways
   to convert the numeric string to integer in C language.

   Example:

     Input: "1234"
     Output: 1234
     Explanation: The string "1234" is converted to the integer 1234.

     Input: "-5678"
     Output: -5678
     Explanation: The string "-5678" is converted to the integer -5678.

Different Ways to Convert String to int in C

   There are 4 methods to convert a string to int which are as follows:

1. Convert String to int Using atoi( )
   The atoi() function in C takes a character array or string literal as an argument and returns
   corresponding value as integer. It is defined in the <stdlib.h> header file. If you observe the name
   atoi() a little closer, you will find out that it stands for ASCII to Integer.

Syntax of atoi()
atoi(str)

   where, str is the numeric string.

C Program to Convert String to int Using atoi( )

// C program to demonstrate the functioning of the
// atoi() function
#include <stdio.h>
#include <stdlib.h>

int main() {
	char *str1 = "141";
	char *str2 = "3.14";

	// Using atoi to convert str1 and str2 to integer values
	int num1 = atoi(str1);
	int num2 = atoi(str2);

	// Performing arithmetic operation
	int sum = num1 + num2;
	printf("The sum of %d and %d is: %d", num1, num2, sum);

	return 0;
}

   Output
The sum of 141 and 3 is: 144

   Time Complexity: O(N)
   Space Complexity: O(1)

2. Convert String to int Using strtol Function
   The strtol (stands for String to Long) function is a similar function to atoi but it provides
   more control and error handling compared to atoi.

Syntax of strtol
strtol(str, &end, base);
     * str: string which represents an integral number.
     * end: string that contains the characters of the original string from the point where the
       conversion stopped. It can be NULL if the string str is completely converted.
     * base: specifies to the base of the integral value.

Program to Convert String to int Using strtol Function

// C program to demonstrate the functioning of the
// strtol() function
#include <stdio.h>
#include <stdlib.h>

int main() {

	char *str1 = "1234";
	char *str2 = "-5678";
	char *endptr;

	// Convert strings to long integers using strtol
	long num1 = strtol(str1, &endptr, 10);
	long num2 = strtol(str2, &endptr, 10);

	// Performing arithematic operations
	long sum = num1 + num2;
	printf("The sum of %ld and %ld is: %ld", num1, num2, sum);

	return 0;
}

   Output
The sum of 1234 and -5678 is: -4444

   Time Complexity: O(N)
   Space Complexity: O(1)

3. Convert String to int Using sscanf()
   We can use sscanf() to easily convert a string to an integer. This function reads the formatted
   input from the string buffer, so we can use the format specifiers to read the numerical
   characters as integers. It is declared inside the <stdio.h> header file.

Syntax
sscanf (source, formatted_string, ...)

   where,
     * source: source string.
     * formatted_string:  a string that contains the format specifiers.
     * ... :  variable arguments list that contains the address of the variables in which we want to
       store input data.

Program to Convert String to int Using atoi( )

// C program to demonstrate how to convert a string
// into a number using sscanf()
#include <stdio.h>

int main() {
	char *str1 = "1234";
	char *str2 = "-5678";


	int num1, num2;
	// Reading the numeric string as integer
	sscanf(str1, "%d", &num1);
	sscanf(str2, "%d", &num2);

	// Performing arithmetic operation
	int sum = num1 + num2;
	printf("Sum of %d and %d is: %d", num1, num2, sum);

	return 0;
}

   Output
Sum of 1234 and -5678 is: -4444

   Time Complexity: O(N)
   Space Complexity: O(1)

     Note: If the text contains other character than numbers, behaviour is undefined.

4. Convert String to int Manually Using Loops
   We can use loops to convert a string to an integer character by character using their ASCII values.
     * Initialize a variable num = 0.
     * Start traversing the string str from the front.
     * For each character ch, check if they are numeric or non-numeric ASCII value using the range of
       numbers i.e. [48, 57].
     * If ch is numeric,
          + Convert it to corresponding integer value using offset (-48) or simply '0'.
          + Add this value to the num * 10 to maintain the digit places.
     * If ch is not numeric, break out of the loop.

Program to Convert String to int Manually Using Loops

// C Program to convert string into integer manually
// using for loop
#include <stdio.h>
#include <string.h>

int main() {
	char *str = "4213";
	int num = 0;

	// Converting string to number manually using loop
	for (int i = 0; str[i] != '\0'; i++) {
		if (str[i] >= 48 && str[i] <= 57) {
			num = num * 10 + (str[i] - 48);
		}
		else {
			break;
		}
	}

	printf("The converted number is: %d", num);
	return 0;
}

   Output
The converted number is: 4213

   Time Complexity: O(N)
   Space Complexity: O(1)

Conclusion
   In C programming, converting a string to an integer is essential when you need to perform numeric
   operations on a value represented as a string. There are multiple methods to achieve this conversion,
   each with its own advantages:
     * Using atoi() Function: A straightforward and commonly used method, but lacks error handling.
     * Using strtol() Function: Provides more control and error handling, making it a robust alternative
       to atoi().
     * Using sscanf() Function: A versatile option that uses formatted input conversion.
     * Manual Conversion Using Loops: Offers complete control over the conversion process but requires
       more effort.
